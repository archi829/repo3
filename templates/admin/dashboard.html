API Reference use the request builder below to test endpoints
{# ── API cheat-sheet ── #}

GET /api/stats

Returns total counts of students, companies, drives, applications.

GET /api/students

Returns list of all students (id, name, email).

POST /api/students

Create a new student. Copy this into the body field:

{
  "full_name": "Alice Kumar",
  "email": "alice@test.com",
  "password": "pass123",
  "cgpa": 8.5,
  "skills": "Python, Flask"
}

PUT /api/drives/<id>

Approve or reject a drive. Copy this into the body field:

{ "status": "Approved" }

{ "status": "Rejected" }

{ "status": "Closed" }

DELETE /api/students/<id>

Delete a student by ID. No body needed.

DELETE /api/drives/<id>

Delete a drive by ID. No body needed.
